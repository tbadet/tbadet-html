This folder holds your images. Add files with these exact names and the
site's <img> tags (once you swap them in for the placeholder <div>s) will
pick them up automatically:

  profile.jpg     — your headshot/portrait, used in the homepage hero
  pub-1.jpg       — thumbnail/cover image for publication 1
  pub-2.jpg       — thumbnail/cover image for publication 2
  pub-3.jpg       — thumbnail/cover image for publication 3
  (add pub-4.jpg, pub-5.jpg, etc. as needed for pages/publications.html)

You can also add a CV PDF here at the repo root as assets/cv.pdf — the
"Download PDF" button on pages/cv.html already links to it.

Recommended sizes: profile.jpg roughly square (e.g. 800x800px),
publication thumbnails roughly 16:10 (e.g. 800x500px). Compress images
before adding them (e.g. with tinypng.com) so the site loads quickly.

Delete this file once your real images are in place — git doesn't track
empty folders, so it exists just to keep this folder in the repo until then.
